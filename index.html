How to Add a New Blog Post

To add a new blog post, follow these steps:

  1. Create a new HTML file in the posts/ directory with a descriptive name (e.g., posts/implementing-aiops-solutions.html)
  2. Add any images to the assets/images/ directory
  3. Update the index.html file to include a link to the new blog post by copying and modifying the template in the comments
  4. Commit and push the changes

The blog post will automatically be deployed to GitHub Pages.
